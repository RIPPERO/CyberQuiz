# CyberQuiz

### 1. Make sure you have installed node and git.

### 2. Clone this repo.

### 3. Hit

```sh
Frontend:
npm install
npm start

Backend:
cd src/Server
node index.js

Database:
Make sure your MySQL database is running, .env.example is renamed to .env and it is configured properly.
```
### 4. You should see your browser opened automatically.
